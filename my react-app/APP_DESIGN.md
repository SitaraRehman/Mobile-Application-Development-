## Design Principles

### Typography
- **Headers**: Bold, 24-28px
- **Subtitles**: Regular, 14-16px
- **Product Names**: Bold, 16px
- **Prices**: Bold, 18px
- **Buttons**: Bold, 14px

### Colors
- **Primary Background**: #f5f6fa (Light Gray)
- **Card Background**: #ffffff (White)
- **Text Primary**: #2c3e50 (Dark Blue-Gray)
- **Text Secondary**: #7f8c8d (Gray)
- **Success/Price**: #27ae60 (Green)
- **Header Bar**: #2c3e50 (Dark Blue-Gray)

### Spacing
- **Card Padding**: 15-20px
- **Screen Padding**: 10-20px
- **Element Margins**: 10-15px
- **Section Spacing**: 15-30px

### Shadows
- **Cards**: Subtle shadow for depth
- **Buttons**: Light elevation
- **Category Cards**: Medium shadow

### Layout
- **Responsive Grid**: 2 columns for categories
- **Product Cards**: Horizontal layout (image left, info right)
- **Full Width**: Buttons and category selectors

### Navigation
- **Stack Navigator**: Native stack for smooth transitions
- **Header**: Consistent across all screens
- **Back Button**: Standard < arrow with screen title

## Responsive Design

The app is designed to work on various screen sizes:

- **Small Phones** (< 360px): Single column layouts
- **Medium Phones** (360-414px): Standard 2-column grid
- **Large Phones/Tablets** (> 414px): Optimized spacing

All dimensions use `Dimensions.get('window')` for dynamic sizing.
